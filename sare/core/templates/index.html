Pescar

O Projeto PESCAR visa realizar um estudo a partir da realidade sócioeconômica de cada estudante do IFTO. Há uma equipe multiprofissional responsável pelo desenvolvimento do projeto, e, nessa primeira etapa, afim de validar as ações de pesquisa e mapeamento de vulnerabilidades, o projeto é desenvolvido por Samuel Barbosa, como Trabalho de Conclusão de Curso do curso Sistemas para Internet, do Campus Palmas (IFTO).
